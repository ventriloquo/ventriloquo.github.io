Já imaginou usar uma ferramenta que é parte de uma linguagem de programação... para fazer um Blog?

Pois é. É possível.

E no caso desse blog, eu utilizo a ferramenta "haredoc", um "gerador de documentação" da linguagem de programação Hare

Para baixar qualquer um dos arquivos dos posts, basta colocar "hare_blog/" no lugar da hashtag e colocar ".ha" no final na barra de endereço!

Aqui vai um exemplo: https://tukain.xyz/hare_blog/post_01_01_1970.ha

Quero também deixar claro que eu não usei puramente o Haredoc para construir este site. Fiz uso de JavaScript em lugares relacionados à Markup e Syntax Highlighting por exemplo, e há também uma Makefile que "compila" os vários arquivos presentes na raíz do código-fonte deste site.
